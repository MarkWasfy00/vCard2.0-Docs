## Where can i modify data

open your folder then go to src/data

here you will see every section inside this folder 



```bash
  ..
├── src/
│   ├── data/
│   │   ├── Info.json
│   │   ├── Resume.json
│   │   ├── Portfolio.json
│   │   ├── Blog.json
│   │   ├── Contact.json
│   │   └── Styles.json
│   

```
